# iFood-SmartOps---Multi-Agent-System-para-Resolu-o-de-Log-stica
iFood SmartOps: Sistema Multi-Agente (LangGraph) para resolução de exceções logísticas. Integra pipelines de PySpark para data profiling, geração de dados sintéticos para corpora de treino e framework de Evals (DeepEval) para métricas de precisão. Arquitetura escalável em FastAPI e pronta para Kubernetes.
